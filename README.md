3. Qvc cancella da solo le righe obsolete (con qty=0 per 3+ mesi) tramite il meccanismo del timestamp
4. Colonne db: ga_code, itemid, qty, extra_qty, prc, extra_prc, notes, qty_changed, prc_changed, timestamp
